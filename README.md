# SeastyleCrawler
## Usage

you confirm configuration file. (./seastyle_crawler.yaml.example)
and override <marina code> , <reserve status api url>
copy to ./seastyle_crawler.yaml

```
$ bundle exec ruby exe/seastyle_crawler -c ./seastyle_crawler.yaml
```
